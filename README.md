Terima kasih telah bergabung bersama komunitas trader cerdas! Untuk menggunakan aplikasi ini, Anda tidak perlu mengerti koding. Ikuti langkah sederhana di bawah ini:

A. Cara Mengaktifkan Aplikasi (Online & Bisa di HP)
Kami menggunakan layanan Vercel (Gratis) agar Anda memiliki link jurnal pribadi yang aman dan bisa diakses kapan saja.

Klik Link Deploy: [MASUKKAN LINK DEPLOY KAMU DI SINI]

Login/Daftar: Gunakan akun Google atau Email Anda untuk daftar di Vercel.

Clone Repository: Masukkan nama proyek Anda (misal: Jurnal-Trading-Saya) lalu klik Create.

Tunggu 1 Menit: Vercel sedang membangun aplikasi Anda. Jika muncul kembang api, selamat! Aplikasi Anda sudah online.

Buka Link: Klik tombol "Go to Dashboard" lalu klik "Visit" untuk membuka jurnal Anda.

B. Cara Memasang di Layar Utama HP (PWA)
Agar terasa seperti aplikasi asli dari Play Store/App Store:

Pengguna Android (Chrome):

Buka link jurnal Anda di Chrome.

Klik titik tiga (â‹®) di pojok kanan atas.

Pilih "Instal Aplikasi" atau "Tambahkan ke Layar Utama".

Pengguna iPhone (Safari):

Buka link jurnal Anda di Safari.

Klik ikon Share (kotak dengan panah ke atas).

Scroll ke bawah dan pilih "Add to Home Screen".

C. Pengaturan Awal (PENTING!)
Agar statistik akurat, lakukan ini pertama kali:

Buka menu Settings.

Isi Nama & Foto Profil Anda.

Tentukan Starting Balance (Modal Awal) Anda.

Daftarkan Trading Pair (misal: XAUUSD, BTCUSDT) dan Strategi yang Anda gunakan.

Data Anda tersimpan secara lokal di browser; jangan lupa lakukan Export CSV di menu Settings secara rutin untuk backup ke Excel.
