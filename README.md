C. Pengaturan Awal (PENTING!)
Agar statistik akurat, lakukan ini pertama kali:

Buka menu Settings.

Isi Nama & Foto Profil Anda.

Tentukan Starting Balance (Modal Awal) Anda.

Daftarkan Trading Pair (misal: XAUUSD, BTCUSDT) dan Strategi yang Anda gunakan.

Data Anda tersimpan secara lokal di browser; jangan lupa lakukan Export CSV di menu Settings secara rutin untuk backup ke Excel.
